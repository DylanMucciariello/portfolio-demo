Storm Surge

Storm Surge is a comprehensive full-stack web application integrated with prototype IoT tracking devices. It aims to streamline and enhance hurricane relief efforts by connecting volunteers, shelters, and donation centers in real time. By providing real-time alerts and a centralized volunteer registration portal, Storm Surge simplifies relief efforts and ensures that all essential information is accessible in one platform.

shrimptype

Shrimp Type is a sleek React and TypeScript web app designed to improve typing skills. It features timed and word-based tests, tracks WPM and accuracy, and visualizes progress with dynamic graphs. With a clean, responsive design, it makes typing practice engaging and insightful.
